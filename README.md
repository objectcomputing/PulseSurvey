# Recreating the ResponseKey.id bug
Clone this repository locally. Then...
## Stand up the PostgreSQL Database
```
docker-compose up
```

## Run the send-surveys tests
```
cd send-surveys
gradlew test
```

## Check the test report
```
open build/reports/tests/test/index.html
```
